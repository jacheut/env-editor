# 环境变量编辑器

一个简洁易用的 Windows 环境变量图形化编辑工具，使用 Python 和 tkinter 开发。

## 功能

- 可视化编辑用户环境变量
- 可视化编辑系统环境变量（需要管理员权限）
- 支持添加、修改、删除环境变量
- 支持刷新查看最新环境变量
- 界面简洁，操作直观

## 运行要求

- Python 3.x
- Windows 操作系统

## 使用方法

```bash
python main.py
```

## 界面说明

- **用户环境变量**：编辑当前用户的环境变量
- **系统环境变量**：编辑系统级环境变量（需要管理员权限）
- **添加**：新增环境变量
- **修改**：编辑选中的环境变量
- **删除**：删除选中的环境变量
- **刷新**：重新加载环境变量列表

## 注意事项

- 修改系统环境变量需要管理员权限
- 修改环境变量后，新打开的程序会生效，已运行的程序可能需要重启
- 请谨慎删除系统环境变量，以免影响系统正常运行
